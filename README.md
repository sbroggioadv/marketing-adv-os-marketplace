# Marketing-Adv-OS — Plugin Claude

Plugin profissional de marketing para escritórios de advocacia e empreendedores jurídicos. **47 skills** cobrindo identidade visual, copywriting, artes, produção e auditoria de compliance (OAB Provimento 205 ou CONAR+LGPD).

## Instalação rápida

**Claude Desktop (Cowork):**
1. Settings → Plugins → Add marketplace
2. URL: `https://github.com/sbroggioadv/marketing-adv-os-marketplace`
3. Selecione **marketing-adv-os** → Install
4. Reinicie e digite `/start-marketing`

**Claude Code (CLI):**
```bash
claude plugin marketplace add sbroggioadv/marketing-adv-os-marketplace
claude plugin install marketing-adv-os@marketing-adv-os-marketplace
```

## O que vem dentro

- 🎨 Identidade visual: paleta, voz, tipografia, tokens CSS
- ✍️ 18 skills de copywriting: headlines AIDA, ads Meta/Google, emails, posts, CTAs
- 🖼️ 6 skills de artes: PNG, PPTX, carrossel, stories, reels
- 📄 5 skills de produção: landing pages, ebooks, slides
- 🔌 5 integrações opt-in: Meta Ads, Canva, Postiz, email broadcast
- ⚖️ 5 skills da Diretoria Criativa R1-R4 — auditoria automática

## Track selector

- **Track A — Advogado/escritório**: OAB Provimento 205/2021 com 15 vedações editoriais
- **Track B — Empresário/criador**: CONAR + LGPD, anti-spam, opt-in obrigatório
