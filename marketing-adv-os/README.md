# marketing-adv-os

Plugin profissional de marketing para escritórios de advocacia e empreendedores jurídicos.

## O que faz

47 skills cobrindo o ciclo completo de marketing — da definição de marca à publicação auditada:

- **Identidade visual:** paleta, voz, tipografia, tokens CSS, mood board
- **Copywriting:** headlines AIDA, ads Meta/Google, emails (welcome/lançamento/upsell), posts sociais, CTAs, storytelling
- **Artes:** PNG, PPTX, carrosséis, stories, reels — com mapa de medidas atualizado 2026
- **Produção:** landing pages (HTML + Next.js 15), ebooks (MD + PDF), slides
- **Integrações opt-in:** Meta Ads, Canva, Postiz, email broadcast
- **Auditoria:** Diretoria Criativa R1-R4 (brief, copy, compliance, performance) com bloqueio automático em violação OAB/CONAR

## Track selector

No primeiro uso, escolha seu perfil:

- **Track A — Advogado/escritório:** governance OAB Provimento 205/2021 + 15 vedações editoriais aplicadas automaticamente em toda peça
- **Track B — Empresário/criador:** governance CONAR + LGPD com anti-spam, opt-in obrigatório e base legal explícita

## Como usar

Após instalação, em qualquer conversa:

```
/start-marketing
```

Wizard de 11 blocos configura sua marca e salva em `<seu-projeto>/marketing/persona.md`.

Depois:

```
/campanha-marketing       # planeja uma campanha
/conteudo-marketing       # gera posts, carrosséis, stories, reels
/copy-marketing           # gera headlines, ads, emails, CTAs
/revisao-marketing-final  # auditoria OAB/CONAR pré-publicação
```

## Privacidade

- Tudo salvo em `<seu-projeto>/marketing/` (gitignored por default)
- Integrações externas são opt-in com warning
- Nenhum dado de cliente é coletado pelo plugin
- Compatível com obrigações LGPD do operador
