# MARKETING COPY ADS META

> Skill Tier 2 (Executor — Copy). Copy para Meta Ads feed (FB+IG) com 9 variacoes.

## OBJETIVO

Gerar 9 copies de anuncio em 3 tamanhos (3 curtas + 3 medias + 3 longas) para Meta Ads, prontas para subir no Gerenciador. Cada copy: texto principal + headline + descricao + CTA + gatilho identificado.

## PRE-REQUISITOS

Workspace + MEMORY com publico+dor+oferta. Track determinado (VE-04 bloqueia certas estruturas em A).

## FLUXO

### 1. Briefing rapido

> "Vou gerar copy pra Meta Ads de: [Oferta].
> Confirme/complete:
> - Objetivo da campanha: conversao / trafego / leads / engajamento / video views
> - Oferta especifica: desconto, bonus, prazo (se houver)
> - Destino do clique: LP, WhatsApp, checkout direto, perfil
> - Posicionamento principal: feed / stories / reels"

### 2. Aplicar voz + regras Meta

**Regras OBRIGATORIAS da Meta (qualquer track):**
- NAO usar "voce" de forma acusatoria ("se VOCE esta gordo")
- NAO mencionar atributo pessoal sensivel (peso, doenca, raca, orientacao)
- NAO prometer resultado absoluto ("garanto que voce vai")
- NAO usar antes/depois exagerado
- Linguagem positiva e aspiracional

**Track A — OAB Provimento 205 (alem das regras Meta):**
- NAO prometer resultado processual (VE-04)
- NAO captacao indevida — VE-06 (sem "ligue agora", "consulte ja", urgencia falsa)
- Conteudo informativo > publicitario

**Track B — CONAR (alem das regras Meta):**
- Se houver comparativo, deve ser real e verificavel (VE-10)
- Parceria paga deve ter #publi (VE-13)

### 4. Recomendacao de teste

> "PLANO DE TESTE SUGERIDO:
> - Subir 3 variacoes (1 curta + 1 media + 1 longa) por audiencia
> - Budget inicial: R$ 30-50/dia por variacao
> - Janela: 5-7 dias
> - Otimizar para [objetivo selecionado]
> - Vencedora: aquela com CTR > [benchmark] e CPL/CPC <= [benchmark]"

### 5. Diretoria Criativa R2 + R3 + R4

R2 (Copy): hook + clareza + CTA singular + gatilho nominado.
R3 (Compliance): regras Meta + VE conforme track.
R4 (Performance): UTM sugerido pra cada variacao, identificacao da variacao no link.

## OUTPUT

9 copies estruturadas + plano de teste + veredito Diretoria Criativa.

## VEDACOES ESPECIFICAS

- **Meta Policy** — NUNCA "voce" acusatorio, atributo pessoal sensivel, promessa absoluta, antes/depois exagerado
- **VE-01** — NUNCA superlativo sem prova
- **VE-04 (Track A)** — NUNCA promessa de resultado processual
- **VE-06 (Track A)** — NUNCA CTA com captacao agressiva
- **VE-12** — NUNCA propaganda enganosa
- **VE-13** — Se ha parceria paga, DECLARAR (#publi ou rotulo equivalente no proprio texto)

## PROTOCOLOS ACIONADOS

- **2.1 Briefing** — objetivo + oferta + destino
- **2.3 Producao** — voz + paleta de gatilhos
- **2.4 Compliance** — regras Meta + VE track-aware
- **2.5 Mensuracao** — plano de teste com benchmark + janela
